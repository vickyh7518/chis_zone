Php session 使用方法

起手式 session_start()

要使用 session ，就需要在一開始呼叫 session_start()，啟動後，就可以使用 $_SESSION 去存取資料

session 儲存位置

php.ini 預設的保存方式是檔案儲存(session.save_handler = files)，會放在 session.save_path 指定的路徑，
